The everyday web still inspires me.

28 Oct 2016

It’s amazing how far the web has come in all that time from being so wild, so insecure, so fresh, to the point that it’s just so banal.

I have no children, which makes it difficult to completely empathise with the child of the First World today. But I’ll bet that, and yes I’m generalising here, that because the web is so ubiquitous , that using it as learning tool has become so mundane and everyday that it doesn’t excite or enthrall them, like it does for me.

I don’t care if anyone thinks its nerdy to love all this stuff. I love it. If I see a new bit of fancy UI, I’ll instantly F12 it just so I can poke around someone’s CSS or javascript. (Of course you could do more delving back in the day before everyone started minifying everything).

It’s hard to imagine the kids of today getting excited about a bit of ajax, to be honest I probably wouldn’t have been excited by ajax when I was nine - I loved to play Spy Hunter when I was nine. I didn’t even know what a modem was, let alone would I know thing about electronic communication.

I always used to try to run over all of the ducks in my speedboat in Spy Hunter

I know stuff like ajax is such a basic, commonplace thing on today’s web. But I still think ajax is cool, it’s still a little bit like magic.
